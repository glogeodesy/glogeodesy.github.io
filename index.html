GLONASS Frequency Channels

Purpose

GLONASS is different from GPS, especially the FDMA. We need to determine the frequecy channels of GLONASS satellites firstly if we want to using GLONASS data to positioning, navigation,timing and do something else such as ionosphere modeling, troposphere monitoring and so on. Obviously, GNSS receiver receives the frequency channels through broadcast ephemeris. We actually don't have to use the navigation file to get frequencies of satellites everytime when data processing. Because the frequency channels won't be changed until the satellite is replaced by a new satellite.

Therefore, a program is developed and running on server. It tracks the official information of GLONASS and generates a file(glo.dat) which records frequency channels of GLONASS satellites. The file is updated everyday automatically. The value is not available if a satellite frequency channel is 99.
